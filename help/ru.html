Приложение загружает активные журналы Windows или файлы EVTX/EVT, создаёт полный Markdown с доказательствами, выполняет локальный анализ Ollama и поддерживает последующие вопросы по событиям и результатам анализа.

Порядок работы

  1. Обновите дерево, выберите журнал и загрузите его.
  2. Задайте лимит; 0 означает все события.
  3. При необходимости используйте XPath-фильтр.
  4. Выберите модель Ollama и язык ответа, затем запустите анализ.
  5. Используйте вкладку диалога для уточняющих вопросов.

Полнота данных

Для каждого события полный Markdown сохраняет сводку, все распрямлённые XML-поля и исходный необработанный XML. PDF является производной сводкой.

Ollama

Адрес по умолчанию: http://127.0.0.1:11434. Потоковые запросы выполняются в фоне, поэтому интерфейс остаётся доступным. Автоматический режим делит большие данные на части и затем создаёт итоговый вывод.

Конфиденциальность

При локальном адресе данные остаются на компьютере. Удалённый адрес получает выбранный диагностический контекст.

Системный контекст и отчёты

Снимок системы можно проверить и отредактировать перед отправкой. Автоматические Markdown- и PDF-отчёты отключаются в настройках. PDF содержит вычисленные таблицы и диаграммы.

# Changelog

## 0.1.0 — 2026-07-10

Initial implementation.

- Event Viewer-style live channel tree from the current Windows system.
- Offline EVTX/EVT loading through `wevtutil`.
- Complete Markdown export with every flattened XML field and raw XML.
- Search, severity filters, complete detail view, and calculated dashboard.
- Local Ollama model discovery, strict actionable diagnostic prompt, streamed non-blocking output, and automatic chunked analysis for large contexts.
- Follow-up chat tab grounded in loaded events, latest analysis, system snapshot, and recent conversation.
- Optional system snapshot and Windows SAPI TTS.
- Automatic Markdown and PDF reports with calculated charts and tables.
- English default plus German, French, and Russian runtime language selection.
- Light, Dark, Sepia, Ocean, Matrix, Hellfire, Purple, and Aurora themes.
- Optional explanatory tooltips and a local Help tab.
- ANSI-colored, repeatable Windows installer with visible version, install logs, wheelhouse support, self-tests, and automatic application start.
